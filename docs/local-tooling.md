# Herramientas locales

Para validar el proyecto localmente se requiere tener disponibles en PATH:

```text
node
npm
```

Versiones recomendadas:

```text
Node.js 22 LTS o superior
npm 10 o superior
```

Comandos de verificacion:

```bash
npm install
npm run typecheck
npm run lint
npm run build
```

En la sesion actual solo esta disponible `node`; por eso no se puede instalar dependencias ni ejecutar build desde este entorno.
